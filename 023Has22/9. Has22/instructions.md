# Instructions  

Given an array of ints, return true if the array contains a 2 next to a 2 somewhere.

  ## Test Cases
  1. has22([1, 2, 2]) -> true
  2. has22([1, 2, 1, 2]) -> false
  3. has22([2, 1, 2]) -> false
